# Local AI Companion (Pi 3 B+ Optimized)

A resource-conscious, stateful, and highly contextual personal AI companion designed to run headless 24/7 on restricted hardware (optimized for Raspberry Pi 3 Model B+ with 1 GB RAM). 

The companion uses **strictly sequential orchestration** to ensure its memory footprint remains under 300 MB, utilizing remote AI endpoints (OpenAI-compatible LLM endpoints and Google Gemini Embeddings) for resource-heavy inference tasks.

---

## Core Features

*   **Hot Short-Term Memory (STM):** An in-memory, token-aware, FIFO buffer restricted to the last 20 messages, avoiding database latency during active chat.
*   **Medium-Term Memory (MTM):** Automatically summarizes your conversations into concise, structured diary entries every 20 messages (or 24 hours).
*   **Long-Term Memory (LTM):** Uses `pgvector` with 16-bit half-precision floating-point vectors (`halfvec(3072)`) to index and perform semantic cosine similarity searches on historical diary entries and messages.
*   **The Soul Map (Facts Extraction):** Background tasks extract permanent facts about you (preferences, routine, relationships, goals) from your diaries, permanently pinning them to the system prompt so the bot never forgets your identity.
*   **Emotional State Tracker:** Analyzes your mood and energy levels every 4 messages to adjust the companion's tone dynamically.
*   **Dynamic Personality Mutation:** Every night at 3:00 AM, the bot goes into "Dream Mode", reflecting on the previous day's events and subtly mutating its own personality traits to better match your current emotional needs.
*   **Sandboxed System Tools:** Equipped with a strictly sequential tool execution loop, allowing the companion to fetch current weather, search the live web via Exa, read/write files, and execute shell commands inside a restricted workspace.
*   **Proactive Pulse (Check-ins):** Kate can autonomously message you during the day if you have been away, referencing your actual goals or recent diary entries to ask how you're doing.

## Prerequisites

*   **Hardware:** Raspberry Pi 3 Model B+ (1 GB RAM), 16GB MicroSD card.
*   **OS:** Raspberry Pi OS Lite (64-bit) recommended.
*   **APIs Required:**
    *   An OpenAI-compatible LLM Provider API Key and Base URL (e.g. NVIDIA NIM, Groq, DeepSeek).
    *   Google Gemini API Key (for `gemini-embedding-2` vector creation).
    *   Exa API Key (for live web searches).
    *   Telegram Bot Token (obtained from [@BotFather](https://t.me/BotFather)).

## Environment Variables (`.env`)

Create a `.env` file in the root directory and configure it as follows:

```bash
# Server Port
PORT=3000

# Google Gemini (Used for pgvector embeddings)
GOOGLE_API=AIza...
EMBEDDING_MODEL=gemini-embedding-2

# OpenAI-Compatible LLM Provider (e.g., NVIDIA NIM, DeepSeek, Together)
LLM_API=your-llm-api-key
LLM_ENDPOINT=https://integrate.api.nvidia.com/v1
LLM_MODEL=meta/llama-3.1-8b-instruct

# Telegram Variables
BOT_TOKEN=123456:ABC-DEF...
ALLOWED_USER_ID=999999999

# External Web Search
EXA_API=exa_your_api_key

# PostgreSQL Database Configuration
PGHOST=127.0.0.1
PGPORT=5432
PGUSER=bot_user
PGPASSWORD=strong_password
PGDATABASE=chatbot_db

# Companion Config
WORKSPACE_DIR=/home/pi/companion_workspace
MAX_TOOL_ITERATIONS=3
PROACTIVE_COOLDOWN_HOURS=3
```

## Installation & Setup

1.  Clone the Repository to your Raspberry Pi:
```bash
    git clone https://github.com/your-username/local-ai-companion.git
    cd local-ai-companion
```

2.  Configure Environment Variables: Copy dummydotenv.txt to .env and fill out
    your API tokens and custom IDs.
```bash
    cp dummydotenv.txt .env
    nano .env
```

3.  Run the Autodeploy Script: Our automated installation script handles
    dependency resolution, compiles the latest pgvector extension supporting
    half-precision vectors, configures system memory limits, injects optimal
    database configurations for 1GB RAM, runs the SQL schema, and registers PM2.
```bash
    chmod +x setup.sh
    ./setup.sh
```

4.  Confirm Autostart on Boot: Configure PM2 to autostart the companion on Pi
    reboot:
```bash
    pm2 startup
    # (Execute the command generated by the output of the line above)
    pm2 save
```

## Commands Guide

Interact with your companion directly on Telegram using these custom commands:

  - `/status` — View uptime, message handle statistics, and real-world system RAM
    footprint.
  - `/remember [text]` — Force-commit an important fact directly into long-term
    vector memory.
  - `/diary` — Read recent medium-term diary entries.
  - `/tools` — Toggle the companion's ability to use web searching,
    writing/reading files, and running shell scripts.
  - `/retry` — Safely re-runs the last message if the API provider times out or
    drops. It can also be used as a "Regenerate Reply" button.
  - `/reset` — Wipe current Short-Term Memory (STM) cache to start a fresh thread.
  - `/help` — Show the command directory.

## Nightly Maintenance Job

Every night at 3:00 AM, a sequential cron pipeline runs to maintain database
health and keep the companion evolving:

1.  **Habit Consolidation:** Connects the dots between recent events and old
    memories to extract behavioral insights.
2.  **Fact Decay:** Slowly drops the confidence of stored facts over time to mimic
    natural forgetting.
3.  **STM Cleanup:** Clears out raw messages older than 14 days to keep the SD card
    from filling up.
4.  **LTM Embedding:** Embeds unprocessed summaries into vector_memories.
5.  **Nightly Reflection:** Drafts an internal monologue summarizing your
    relationship progress.
6.  **Personality Mutation:** Adjusts Kate's core temperament based on how you have
    been doing emotionally.
